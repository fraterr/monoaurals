# Monoaurals

Monoaurals is a premium, web-based real monaural beat generator and brainwave entrainment application. Unlike binaural beats, which require headphones to deliver different frequencies to each ear, monaural beats are created by physical wave interference directly in the audio signal itself. This physical amplitude modulation is highly effective whether listening through external speakers or headphones.

## Features

- **Brainwave Presets**: Quick-select presets for Delta, Theta, Alpha, Beta, and Gamma wave states.
- **Precision Modulation Controls**: Fine-tune the stimulus (beat) frequency and adjust the base carrier frequency to your preference.
- **Session Timer**: Select standard durations or specify a custom time, complete with a built-in 3-minute linear fade-out to prevent abrupt transitions.
- **Audio Mixer**: Mix the main beat tone, background pink noise, and master volume to suit your listening environment.
- **Real-Time Visualizer**: Oscilloscope canvas displaying the actual wave interference pattern generated by the audio synthesis.
- **Direct Local Export**: Synthesize and download the entire session directly in your browser as a lossless `.wav` or a compact `.mp3` file.

## Tech Stack

- **Core**: Vanilla HTML5, CSS3, and JavaScript (ES6+).
- **Acoustic Engine**: Web Audio API for real-time oscillator synthesis and audio routing.
- **MP3 Encoding**: `lamejs` for efficient client-side audio compression.
- **Styling**: Modern dark mode design featuring glassmorphism and subtle animations.

## Getting Started

1. Clone or download the repository.
2. Open `index.html` directly in any modern web browser, or serve the directory using a local development server:
   ```bash
   # Using python
   python -m http.server 8000
   
   # Or using Node.js
   npx serve .
   ```
3. Customize your frequencies or select a brainwave preset, press **Play**, and relax.
